# 424-remote-controller

## Установка:

### создать виртуальное окружение на питоне:
```bash
python -m venv venv
```
### активировать его:
для Windows:
```bash
.\venv\Scripts\activate
```
для Unix:
```bash
source venv/Scripts/activate
```
### установить необходимые библиотеки:
```bash
pip install -r requirements.txt
```
